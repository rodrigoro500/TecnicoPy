# Arquitectura Oficial de TécnicoPy

Versión: 1.0

---

# Filosofía

TécnicoPy no es un simple directorio de profesionales.

Es una plataforma diseñada para generar confianza entre personas mediante tecnología.

Cada componente, pantalla y funcionalidad debe responder una pregunta:

> ¿Esto aumenta la confianza del usuario?

Si la respuesta es no, la funcionalidad debe replantearse.

---

# Principios de Ingeniería

## 1. Componentes pequeños

Cada componente debe cumplir una única responsabilidad.

No construiremos componentes gigantes.

---

## 2. Reutilización

Todo componente debe poder reutilizarse en distintas pantallas.

---

## 3. Diseño consistente

Toda la aplicación debe compartir:

* colores
* tipografía
* sombras
* espaciados
* animaciones

---

## 4. ORION es un sistema

ORION no es un chatbot.

ORION es el sistema inteligente de TécnicoPy.

Puede recomendar.

Puede analizar.

Puede explicar.

Pero nunca reemplaza la decisión del usuario.

---

## 5. El Perfil Certificado es un producto

Cada profesional posee una página profesional propia.

Su perfil representa su reputación digital.

---

## 6. El mapa representa la cobertura nacional

El ParaguayMap no es una decoración.

Es una herramienta para descubrir profesionales.
